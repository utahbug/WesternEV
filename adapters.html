Adapters

Most newer EVs use either NACS or CCS for fast charging. Adapters can help in some cases, but the vehicle, station, charging network, and approved adapter all have to match.

Common connector options

  • NACS: Tesla-style connector now used by Tesla and adopted by many newer North American EVs. Some non-Tesla vehicles still need an approved adapter or network access before using certain Tesla chargers.
  • CCS: Common fast-charging connector on many non-Tesla EVs. CCS vehicles may need an approved NACS adapter to use compatible Tesla Superchargers.
  • J1772: Common Level 2 connector for slower charging at homes, hotels, workplaces, hospitals, and parking lots. It is not the same as DC fast charging.
  • CHAdeMO: Older fast-charging connector used by some older EVs. It is less common on newer public fast-charging routes.
